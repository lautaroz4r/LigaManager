## Descripción general del sistema

LigaManager es un sistema desarrollado en Python por consola que permite gestionar un torneo deportivo amateur. El programa permite registrar equipos, cargar resultados de partidos, calcular automáticamente puntos, diferencia de goles y generar una tabla de posiciones básica.

El objetivo del sistema es facilitar la administración de un torneo, evitando cálculos manuales y permitiendo consultar la información principal de manera ordenada.

## Funcionalidades principales

* Registro de equipos.
* Visualización de equipos cargados.
* Carga de resultados de partidos.
* Cálculo automático de puntos.
* Cálculo de partidos ganados, empatados y perdidos.
* Cálculo de goles a favor, goles en contra y diferencia de goles.
* Generación de tabla de posiciones.
* Estadísticas básicas del torneo.
* Validación de datos ingresados por el usuario.

## Instrucciones de ejecución

2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo principal con Python:
